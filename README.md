Descripción general del sistema

Este proyecto corresponde a la actividad sumativa de la Semana 3 de la asignatura Desarrollo Orientado a Objetos I.
El sistema fue desarrollado en Java para representar información asociada a una agencia de turismo llamada Llanquihue Tour. Para ello, se aplicaron conceptos fundamentales de POO, tales como encapsulamiento, herencia y composición.
El proyecto permite modelar personas vinculadas a la organización mediante clases relacionadas, mostrando la información de distintos guías turísticos a través de la consola.

Descripción de clases

Persona

* Clase base del sistema.
* Contiene rut, nombre y dirección.

Direccion

* Clase utilizada para representar la dirección de una persona.
* Implementa la relación de composición.

GuiaTuristico

* Hereda de Persona.
* Incorpora idioma y años de experiencia.

Main

* Clase principal encargada de crear objetos y mostrar la información en consola.
